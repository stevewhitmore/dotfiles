# dotfiles
My xfce dotfiles for when I inevitably distro hop

### Theme
* Desktop: Arc Dark
* Window Manager: Numix
* Icons: Yltra Flat Arch Blue
* Wallpaper: GLaDOS Minimal

### Directory Structure
* Tmux and Vim: `~` 
* Desktop & Icon themes: `~/.local/share/` 
* Window Manager: `/usr/share/themes/`
